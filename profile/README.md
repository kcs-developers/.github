## 👋 About
본 프로젝트는 개발자들을 위한, 개발자들에 의한 웹 서비스로서, CS 관련 문제풀이 및 멘토링 및 스터디를 위한 화상채팅 서비스를 제공합니다.


## 🌟 Main function
- WebRTC를 활용하여 멘토, 멘티간의 멘토링이나 팀 구성원들간의 스터디룸을 제공하기 위한 화상 채팅 서비스
- 개발과 관련된 문제를 만들고 풀며 개발자로의 성장을 도모하는 문제 풀이 서비스
- 자유롭게 소통할 수 있는 커뮤니티 서비스


## 🛠️ Skills
- Frontend: React
- Backend: Spring Boot, Java, Gradle, Spring Security, OAuth2 Spring Data JPA, Lombok, Spring Rest Docs, JUnut5, Mockito
- DataBase: MariaDB, Redis
- CI/CD: Jenkins, ArgoCD
- Cloud: AWS EKS, EC2, S3, CLoudWatch,
